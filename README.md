# Electronic E-Commerce Store

Basic starter source code for Web Application Development Group 3.

## Technology Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

## Main Modules
- Authentication and Account Management
- Product Browsing and Search
- Shopping Cart and Checkout
- Admin Product and Order Management

## How to Run
1. Copy this folder into `htdocs` if using XAMPP.
2. Start Apache and MySQL in XAMPP.
3. Open `http://localhost/electronic-ecommerce-store/` in the browser.
4. Import `db/schema.sql` into phpMyAdmin later when database work starts.

## Notes
This is a blank starter structure. The pages contain placeholder layout only and can be edited by each group member.
